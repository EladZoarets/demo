# 📂 Project DNA
- **Frontend Stack:** React (Vite), TypeScript, Tailwind CSS.
- **Backend Stack:** Python (FastAPI), Pydantic, Uvicorn.
- **Standards:** - Frontend: Functional components, Lucide icons.
  - Backend: RESTful API, Type hinting, Clean Architecture.
- **Testing:** Vitest (Frontend), Pytest (Backend).
- **Goal:** A modern fullstack application with seamless React-Python integration.

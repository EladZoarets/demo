# 📂 Project DNA

This is the **Amp AI-SDLC Framework** — a configuration-only repository that transforms Amp into a structured, role-based engineering team.

## Stack Context (customize for your project)

- **Frontend Stack:** React (Vite), TypeScript, Tailwind CSS.
- **Backend Stack:** Python (FastAPI), Pydantic, Uvicorn.
- **Standards:**
  - Frontend: Functional components, Lucide icons.
  - Backend: RESTful API, Type hinting, Clean Architecture.
- **Testing:** Vitest (Frontend), Pytest (Backend).
- **Goal:** A modern fullstack application with seamless React-Python integration.

## SDLC Skills

This project provides Amp skills that implement an autonomous SDLC pipeline:

| Skill | Purpose |
|-------|---------|
| `sdlc-orchestrator` | Runs the full PLAN → SCORE → EXECUTE → CHAOS → REVIEW pipeline |
| `sdlc-architect` | Designs solutions, maps files, scores complexity |
| `sdlc-product` | Defines requirements and acceptance criteria |
| `sdlc-manager` | Decomposes work into atomic PR-sized tasks |
| `sdlc-developer` | Implements code and tests |
| `sdlc-chaos` | Stress tests and finds edge cases |

## Usage

Load the orchestrator skill and provide a feature request:

```
Load the sdlc-orchestrator skill, then orchestrate: [your feature description]
```

Or use individual skills for specific phases of the workflow.
